## Prerequisites

- [.NET 9 SDK]
- [Flutter SDK 3.35.5 or higher]

## Running the Application

### 1. Start the Backend


cd backend/BankingBackOffice.Api
dotnet run


The API will start on `http://localhost:5000`

### 2. Run the Frontend


cd frontend
flutter run -d chrome
